# Osnovni Graficki Oblici

Podesavanje prozora programa:

  + velicina
  + Naslov
  + Boja Pozadine

Kako da crtamo:

  + Point
  + Line
  + Recteangle
  + Circle
  + Ellipse
  + Arc
  + Text
  + Polygon
  + Polyline

Kako koristiti boju za:

  + crtanje (pen)
  + bojenje pozadine oblika
  + bojenje pozadine prozora programa
  + boja fonta
  + RGB i ARGB
  + Snimanje izgleda programa
